Kakao Social Login Test Page

Click the button below to start the Kakao login process.

Login Flow:

  1. Clicking the button will take you to Kakao's login page.
  2. After you log in and consent, Kakao will redirect you to our backend's callback URL.
  3. Our backend will process the login, create an account if needed, and issue JWT tokens.
  4. Since this is a simple test page, the final tokens will be displayed as a JSON response in your browser. A real frontend would handle this response and store the tokens.
